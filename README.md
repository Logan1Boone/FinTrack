# FinTrack

FinTrack is a desktop application designed to help users manage their personal finances by tracking income, expenses, budgets, and generating reports.

---

## 📌 Features

- Track income and expenses  
- Manage budgets by category  
- View financial summaries and reports  
- Export reports in CSV and PDF formats  
- Save and load data using JSON or XML files  

---

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:

- Windows PC  
- .NET 8 SDK: https://dotnet.microsoft.com/  
- Visual Studio Code or Visual Studio  

---

## ▶️ Running the Application

1. Open the FinTrack project folder.
2. Run the application from your IDE.
3. The main dashboard will open automatically.

---

## 🖥️ Application Overview

### Dashboard

The dashboard provides a quick overview of your financial status, including:

- **Total Income**
- **Total Expenses**
- **Remaining Balance**

---

### Transactions

Use the **Transactions** tab to manage financial activity.

#### Add a Transaction

1. Enter a description.
2. Enter an amount.
3. Select a category.
4. Choose **Income** or **Expense**.
5. Click **Add Transaction**.

#### Delete a Transaction

1. Select a transaction from the list.
2. Click **Delete Transaction**.

---

### Budgets

Use the **Budgets** tab to set and manage spending limits.

#### Create a Budget

1. Enter a category.
2. Enter a budget amount.
3. Click **Save Budget**.

#### Budget Alerts

- A warning is displayed when expenses exceed the set budget limit.

---

### Reports

Generate and export financial summaries using the **Reports** tab.

#### Available Options

- Generate Report  
- Export CSV  
- Export PDF  

---

## 💾 Data Storage

FinTrack supports saving and loading data using:

- JSON files  
- XML files  

Saved data can be reloaded in future sessions.

---

## 🛠️ Troubleshooting

### Application Will Not Start

- Verify that the .NET 8 SDK is installed.
- Rebuild the project in your IDE.

### Data Not Saving

- Check file and folder permissions.
- Confirm the correct save path is being used.

### Invalid Amount Error

- Ensure only numeric values are entered in amount fields.

---

## ✅ Best Practices

- Update transactions daily  
- Review budgets weekly  
- Export monthly reports  
- Regularly back up saved files
